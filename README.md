# Random Password Generator 🔐

A simple and efficient Python script that generates strong random passwords.

## 📋 Description

This project generates a secure random password using a combination of uppercase letters, lowercase letters, digits, and special characters. It is built as part of the **DecodeLabs Internship** task.

## ✨ Features

- Generates random password of **length 8**
- Includes uppercase, lowercase letters, digits, and special characters
- Simple command-line tool

## 🛠️ Technologies Used

- Python 3
- `random` module
- `string` module
